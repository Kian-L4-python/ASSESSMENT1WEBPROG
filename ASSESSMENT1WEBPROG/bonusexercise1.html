HTML Quick Reference Guide

Headers

Headers provide structure and hierarchy to your content.


 <h1>Heading 1</h1>
 <h2>Heading 2</h2>
 <h3>Heading 3</h3>
 <h4>Heading 4</h4>
 <h5>Heading 5</h5>
 <h6>Heading 6</h6>

Paragraphs

Paragraphs are used for regular text content.


 <p>This is a paragraph.</p>

Lists

Lists are used to specify lists of information.


 <ul>
     <li>Item 1</li>
     <li>Item 2</li>
 </ul>
 
 <ol>
     <li>Step 1</li>
     <li>Step 2</li>
 </ol>

Links

Links are used to link other web pages to a document.


 <a href="https://example.com">Link Text</a>

Images

Display images on your webpage.


 <img src="image.jpg" alt="Description">

Forms

Collect user input with forms.


 <form action="/submit" method="post">
     <input type="text" name="name" placeholder="Name">
     <button type="submit">Submit</button>
 </form>

Tables

Create structured data tables.


 <table>
     <tr>
         <th>Header 1</th>
         <th>Header 2</th>
     </tr>
     <tr>
         <td>Data 1</td>
         <td>Data 2</td>
     </tr>
 </table>

Divs

Generic container for grouping similar contents or elements.


 <div>
     Content here
 </div>
